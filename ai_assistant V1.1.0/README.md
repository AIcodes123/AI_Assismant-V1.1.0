# AI助手应用

一个功能完整的本地AI助手，支持对话、图片生成、搜索等功能。

## 主要功能

✅ **智能对话** - 深度思考，上下文感知
✅ **图片生成** - 稳定扩散模型，408x408分辨率
✅ **网络搜索** - 模拟搜索功能
✅ **图形界面** - 现代化PyQt5界面
✅ **本地运行** - 无需API密钥
✅ **自动保存** - 对话历史持久化
✅ **轻量化** - 每个文件<300行
✅ **进度显示** - tqdm启动进度条
